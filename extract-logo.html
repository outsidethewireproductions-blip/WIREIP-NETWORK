⚜ Extract OTW Logo from Black Paper

This is a one-time helper. Drop your Black Paper PDF below. The script renders page 1, finds the densest gold-pixel cluster (the cover logo), crops it square, and lets you download it as otw-logo.png. Save the download into the project folder, push to GitHub, and the main app's auto-discovery picks it up everywhere.

After download:
  1. Move the downloaded otw-logo.png into WireIP Network DAG/ (next to wireip-dag.html)
  2. Run: git add otw-logo.png && git commit -m "logo" && git push
  3. Vercel redeploys, the main app auto-discovers the file, and shows it in the brand mark + splash everywhere (desktop + mobile)
